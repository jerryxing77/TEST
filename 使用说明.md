# 股票分析系统：小白使用说明

这份说明只讲怎么操作，不讲代码。

## 第一次使用：照着复制这 3 行

先进入项目目录：

```bash
cd /workspace/TEST
```

然后运行简单操作版：

```bash
python run.py
```

看到问题后，如果你只是想试试看，全部输入 `1` 然后回车即可。

完成后，屏幕上会出现类似下面两行：

```text
snapshot=data/20260619T151352Z_premarket.json
report=reports/20260619_premarket.md
```

你只需要打开 `report=` 后面的文件，也就是 `reports/日期_premarket.md` 或 `reports/日期_close.md`。

## 我每天该怎么用？

### 开盘前

运行：

```bash
python run.py
```

第一个问题选：

```text
1. 盘前报告
```

第二个问题建议先选：

```text
1. 演示数据
```

如果你已经想尝试真实行情，再选：

```text
2. Yahoo 行情
```

### 收盘后

运行：

```bash
python run.py
```

第一个问题选：

```text
2. 收盘报告
```

第二个问题同样可以先选演示数据，确认流程会用了再选 Yahoo 行情。

## 报告在哪里？

报告在这个文件夹：

```text
reports/
```

文件名大概长这样：

```text
20260619_premarket.md
20260619_close.md
```

- `premarket` 表示盘前报告。
- `close` 表示收盘报告。

## 怎么改成我自己的股票？

打开这个文件：

```text
config/watchlist.json
```

找到这一行：

```json
"symbols": ["AAPL", "MSFT", "NVDA", "SPY", "QQQ"]
```

把里面的股票代码改成你自己的。例如：

```json
"symbols": ["TSLA", "AMD", "META", "GOOGL", "AMZN"]
```

保存文件后，再运行：

```bash
python run.py
```

## 不想选择，直接一条命令生成

盘前报告：

```bash
python -m stock_analyzer --session premarket --provider demo
```

收盘报告：

```bash
python -m stock_analyzer --session close --provider demo
```

如果想尝试真实行情，把 `demo` 改成 `yahoo`：

```bash
python -m stock_analyzer --session premarket --provider yahoo
```

## 你只需要记住

1. 先运行 `python run.py`。
2. 不懂就都选 `1`。
3. 最后打开屏幕上 `report=` 后面的文件。
4. 想改股票，就改 `config/watchlist.json` 里的 `symbols`。
